核心逻辑：

```csharp
{
    private static VirtualKeyboardManager _instance;
    private VirtualKeyboardForm _keyboardForm;
    private Control _targetControl;  // 当前目标输入控件
    
    public static VirtualKeyboardManager Instance { get; }
    
    // 显示键盘
    public void ShowKeyboard(Control targetControl = null);
    
    // 隐藏键盘
    public void HideKeyboard();
    
    // 切换布局

    // 设置布局模式

```

焦点管理实现：

```csharp
// 在应用程序启动时注册全局钩子
public void Initialize()
{
    Application.Idle += OnApplicationIdle;
}

private void OnApplicationIdle(object sender, EventArgs e)
{
    var focused = Control.FromHandle(GetFocusedControlHandle());
    if (focused != _targetControl && IsInputControl(focused))
    {
        _targetControl = focused;
    }
}

// 判断是否为可输入控件
private bool IsInputControl(Control control)
{
    return control is TextBoxBase || 
           control is NumericUpDown ||
           control is ComboBox ||
           control is MaskedTextBox;
}
```

输入分发：

```csharp
private void SendKeyToTarget(string key)
{
    if (_targetControl == null) return;
    
    if (_targetControl is TextBoxBase textBox)
    {
        // 处理特殊键
        if (key == "Backspace")
        {
            if (textBox.SelectionLength > 0)
                textBox.SelectedText = "";
            else if (textBox.SelectionStart > 0)
                textBox.Text = textBox.Text.Remove(textBox.SelectionStart - 1, 1);
        }
        else if (key == "Enter")
        {
            // 触发 Leave 事件或移动到下一个控件
            SendKeys.Send("{TAB}");
        }
        else if (key == "Space")
        {
            textBox.SelectedText = " ";
        }
        else
        {
            // 插入字符
            textBox.SelectedText = key;
        }
    }
}
```

## 关键技术点

### 1. 跨窗体焦点追踪

使用 `Application.Idle` 事件 + `GetForegroundWindow` API 检测当前焦点控件：

```csharp
[DllImport("user32.dll")]
static extern IntPtr GetForegroundWindow();

[DllImport("user32.dll")]
static extern IntPtr GetFocus();
```

### 2. 输入分发兼容性

支持多种输入控件：
- TextBox / RichTextBox（TextBoxBase）
- NumericUpDown
- ComboBox（可编辑模式）
- MaskedTextBox
- 自定义输入控件（通过事件扩展）

### 3. 触摸优化

- 按键最小尺寸：40x40 像素
- 字体大小：14-18pt
- 按下反馈：颜色变化 + 轻微缩小
- 间距：2-3 像素

### 4. 布局管理

使用 TableLayoutPanel 或 FlowLayoutPanel 动态生成按键布局，确保自适应窗口大小。
